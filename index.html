ContextMenuEditor

Редактор контекстного меню Проводника Windows

Полный контроль над контекстным меню — добавляйте, изменяйте и удаляйте пункты меню для файлов, папок и фона рабочего стола. Все изменения применяются сразу без перезагрузки.

Обычные пункты с командами
Каскадные подменю (любой вложенности)
Разделители для группировки
Настраиваемые иконки (.ico, .exe, .dll)
Условия: файлы, папки, фон, расширения
Передача выделенного пути (%1)
Интуитивный интерфейс (TreeView)
Немедленное уведомление Проводника
🛡️ Требуются права администратора для изменения реестра (HKEY_CLASSES_ROOT).
💻 Система: Windows 7/8/10/11 (64/32 бит), .NET Framework 4.8.

📌 Инструкция

  • Запустите программу от имени администратора.
  • Используйте кнопки Добавить / Изменить / Удалить для настройки пунктов.
  • Для вложенных пунктов сначала создайте подменю, затем добавляйте дочерние элементы, выбрав родителя.
  • Для передачи файла в команду используйте "%1" в строке аргументов.
  • Иконку можно указать как путь к .ico или к .exe,.dll с индексом (например, imageres.dll,3).

⚡ После нажатия «Скачать .msi» вы получите установочный пакет, который разместит программу в Program Files и добавит запись в «Установку и удаление программ».
